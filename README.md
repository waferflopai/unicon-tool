# Unicon – Unicode to Favicon Converter

**Convert any Unicode symbol, emoji, or glyph into a custom favicon.**  
Download as PNG, ICO, or SVG with full transparency support. Live preview, contrast check, and ready-to-use HTML snippet included.

![Version](https://img.shields.io/badge/version-2.1-blue) ![License](https://img.shields.io/badge/license-MIT-green)

---

## Features

- **Live preview** – See your favicon update as you type
- **Multiple export formats** – PNG, ICO (multi‑size), and SVG
- **Flexible styling** – Background color, opacity, shape (square, rounded, circle), symbol color, and font size
- **Transparency support** – Toggle transparent background for use on any site
- **Contrast check** – Warns when symbol colour may be hard to read
- **One‑click presets** – Quick colour themes
- **Copy‑ready HTML** – Generates a `<link rel="icon">` snippet with embedded data URI
- **Symbol gallery** – Quick pick from a set of commonly used symbols

---

## Getting Started

Unicon is a **single HTML file** – no build tools, no dependencies.  
Just open `index.html` in your browser and start converting.

### Option 1: Use directly
- Download the latest `index.html` from the [releases](../../releases) or clone the repo.
- Open it in any modern browser (Chrome, Firefox, Edge, Safari).

### Option 2: Host it yourself
The tool works equally well as a static page. Upload the file to any web server or use services like GitHub Pages.

---

## Usage

1. **Enter a symbol** – type or paste any Unicode character (emoji, glyph, or letter).  
2. **Customize** – choose background, symbol colour, size, shape, and font scale.  
3. **Toggle transparency** – switch on a fully transparent background.  
4. **Download** – click PNG, ICO, or SVG to save your favicon.  
5. **Copy HTML** – embed the generated favicon into your website with one click.  

The preview updates instantly, and a contrast warning appears if the chosen colours may be difficult to read.
